Version 1

Name of doctor/index entered first. Check boxes for each of the 8 vaccines/combo and for each checked then only those vaccines can be entered via loop.

Version 2

Vaccines may not necessarily be entered in order so there are no vaccine checkboxes. Vaccine and vaccine details are selected from drop-down and the total is entered via text box.

The new entry appears on a 'Doctor/index summary page' with a radio button option to add another. Prototype build so each new selection is added to summary list (up to 3 entries).

When all vaccines have been entered there is a summary page for that doctor with a radio button option to enter another doctor and that takes the user round the same loop.

When all doctors/vaccine details have been entered then a summary page is displayed for all doctors and vaccines administered. Submit, Declation and Done.

No vaccine checkboxes.

When a vaccine is selected it appears below selection dropdown with a question to add another vaccine.

When all vaccines have been entered there is a summary page for that surgery with a question to add another doctor.

Pagination for more than 10 entries.

Submit, Declation and Done.
